3n + 1

Collatz conjecture

Input any positive integer. If the number is odd, it is multiplied by 3 and then 1 is added. If the number is even, it is divided by 2.
The process repeats until the number either reaches the 4-2-1-4-loop, or it reaches a different loop, or it approaches infinity.
